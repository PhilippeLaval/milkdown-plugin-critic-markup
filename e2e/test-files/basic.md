# Basic CriticMarkup - All 5 Types

This file demonstrates all five CriticMarkup constructs.

## Insertion

The quick brown fox {++jumped over++} the lazy dog.

## Deletion

The quick brown {--and lazy--} fox ran fast.

## Substitution

The {~~quick~>swift~~} fox ran through the forest.

## Highlight

The fox was {==remarkably fast==} that day.

## Comment

The fox escaped successfully.{>>Is this realistic? Foxes are fast but this seems exaggerated.<<}
